
# Weather-Based Tennis Play Prediction Using Machine Learning Algorithms

## 📌 Project Overview
This project uses Machine Learning techniques to predict whether tennis can be played based on weather conditions. The model analyzes different weather attributes such as outlook, temperature, humidity, and wind to determine if a tennis match can be played.

## 🎯 Objective
The goal of this project is to apply classification algorithms to analyze weather data and make predictions about playing tennis.

## 📊 Dataset
The dataset contains the following features:

- Outlook (Sunny, Overcast, Rain)
- Temperature
- Humidity
- Wind
- Play Tennis (Yes / No)

These attributes are used to train and evaluate the machine learning model.

## ⚙️ Technologies Used
- Python
- Jupyter Notebook

- Matplotlib

## 🤖 Machine Learning Algorithms
The following classification algorithms are used in this project:

- Decision Tree
- Naive Bayes
- Logistic Regression (optional)
